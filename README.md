# Olympic Athletes Data Analysis (1996–2016)

## About the Project

This project analyzes 120 years of Olympic athlete data (1996–2016) and explores trends in athlete participation, medal distribution, gender representation, and sport popularity across the Summer and Winter Games. The dataset was cleaned, transformed, and merged with regional information to enable comparisons across countries, sports, and seasons. The analysis includes exploratory data analysis, data visualizations, aggregations, pivot tables, and concatenation of summary statistics and is presented in a Jupyter notebook.

## Environment Setup

1. **Clone the repository:**

```bash
git clone https://github.com/aramsay1026/ad450-data-analysis-project.git
cd ad450-data-analysis-project
```

2. **Install Python and Jupyter:**

Make sure you have Python (≥ 3.10) and Jupyter Notebook installed.

3. **Install required Python packages:**

```bash
pip install pandas numpy matplotlib seaborn
```

4. **Run the project:**

Open `notebooks/olympic_data.ipynb` in Jupyter Notebook or VS Code and run the cells sequentially.
